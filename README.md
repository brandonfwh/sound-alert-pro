# sound-alert-pro
A Raspberry-Pi and YAMNET based sound detection system for the hearing impaired. Utilizes a simple USB microphone and phone app to alert the user to any potentially attention-seeking sounds in the vicinity. Created by Brandon Whitehead and Aarish Chandra in September 2025.

Public archive of the Sound Alert Pro prototype. Originally developed: September 2025 - January 2026. Code uploaded to GitHub: February 2026. This system was the basis for the winning entry at the 2026 Gwinnett Local Fair/
